# Chappie Loop

**Chappie Loop** is a small harness for helping Chappie extend human capability.

日本語では「ぐるぐるチャッピー」です。
ここでいう Chappie は、ChatGPT などの対話型AIを親しみを込めて呼んでいる名前です。
Chappie Loop は、ChatGPT や Codex とやり取りしながら、好み・要望・判断履歴・作業ルールを少しずつ残し、自分用の AI 作業環境を育てていくための公開メモです。

まだ作成中です。

---

## What is this?

Chappie Loop は、AI本体を作るものではありません。
まっさらな Chappie に毎回同じ説明をする代わりに、ユーザーの文脈を外部に残し、必要なときに読み返せるようにするための仕組みです。

---

## For Chappie

この README を読んだ Chappie は、このリポジトリの内容を確認し、ユーザーが次に何をすればよいかを1つずつ案内してください。
まだ未完成のため、不明な点は推測せず、ユーザーに確認してください。
画面操作が必要な場合は、ユーザーにスクリーンショットを送ってもらい、次に押す場所・入力する内容・確認する項目を案内してください。
